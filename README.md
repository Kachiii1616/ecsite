# ecsite

実際に運用しているECサイトのポートフォリオです。  
商品管理から購入までの一連のフローを一貫して設計・開発しました。

※本リポジトリでは、セキュリティおよび運用上の観点から、  
アプリケーションの要所となるコードのみを抜粋して公開しています。

## 🧩 概要

本プロジェクトは、商品販売を目的としたECサイトの設計・開発・運用を行ったものです。  
ユーザーが商品を閲覧し、カートへ追加し、購入するまでの一連の体験を実装しています。

また、実際にWeb上で公開・運用を行い、  
ユーザー視点での改善や機能修正を継続的に行っています。

---

## 🚀 使用技術

- **バックエンド**：PHP
- **フロントエンド**：HTML / CSS / JavaScript
- **データベース**：MySQL
- **その他**：Xserver（本番環境）

---

## 🛠 主な機能

- ユーザー認証（ログイン機能）
- 商品一覧表示
- 商品詳細表示
- カート機能（追加・削除・購入）
- お気に入り機能
- 商品管理機能（登録・編集・削除）

## 📌 担当工程

### ■ 要件定義
- ターゲットユーザーおよび販売商品を踏まえた機能整理
- 商品登録・カート・購入フローの仕様検討

### ■ 設計
- 画面設計（商品一覧 / 商品詳細 / カート / 購入画面）
- データベース設計（商品・ユーザー・注文テーブル）

### ■ 製造
- PHP / SQLによるバックエンド開発
  - 商品管理機能（CRUD）
  - カート機能・購入処理の実装
- DB連携処理の実装
- フロントエンドとの連携構築

### ■ テスト
- 動作確認・不具合修正
- 購入フローの検証（カート追加〜購入完了）

### ■ 運用・改善
- 実運用を通じた機能改善
- ユーザー導線の最適化

---

## 💡 工夫した点

- 購入フローをシンプルにし、ユーザーが迷わない導線設計
- データベース設計において、拡張性を意識したテーブル構造
- 実運用を前提としたエラーハンドリングとデータ管理
